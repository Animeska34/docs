ISaveable interface used by Saveable to serialize/deserialize MonoBehaviours attached to Saveable GameObjects.

public interface ISaveable

Remarks

This interface is implemented by MonoBehaviours that need custom serialization and deserialization logic when attached to Saveable GameObjects. Saveable objects use this interface to interact with attached components during the game's saving and loading processes.
